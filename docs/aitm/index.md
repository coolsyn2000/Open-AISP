# AITM

AITM will provide AI-based tone mapping for converting linear or HDR representations into display-ready images.

The planned module will focus on tone compression, local contrast, color appearance, and perceptual quality while keeping the output compatible with downstream visualization and evaluation tools.

This module is currently a placeholder. Future documentation should cover:

- input color space
- tone mapping target
- model architecture
- training losses
- qualitative and quantitative evaluation
