# HDR

HDR will host high dynamic range reconstruction and fusion components.

The planned scope is to combine multiple exposures or RAW frames into a wider dynamic range representation while preserving highlight detail, suppressing noise in dark regions, and keeping color consistent with the rest of the Open-AISP pipeline.

This module is currently a placeholder. Future documentation should cover:

- input and output conventions
- exposure alignment
- fusion strategy
- training data generation
- evaluation metrics
